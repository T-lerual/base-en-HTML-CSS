l ecole est bien

Ubuntu a été présenté pour la première fois le 8 juillet 2004 par Mark Shuttleworth et sa société Canonical Ltd. En fait, le projet a été trouvé par quelques programmeurs qui ont été déçus du fonctionnement du distributeur Debian GNU/ Linux. Quatre ans après avoir créé un groupe multidisciplinaire, les programmeurs ont reçu un financement venant de l’entrepreneur sud-africain Mark Shuttleworth.

Certains utilisateurs préfèrent utiliser à la fois Ubuntu et Windows. Dans ce cas, il est recommandé d'utiliser un WSL (Windows Subsystem for Linus). Cela s'appelle également un double démarrage. Cela permet aux utilisateurs d'utiliser Ubuntu en tant qu'application sur Windows. Cela nécessite un ordinateur avec des performances élevées, sinon le fonctionnement de tous les logiciels Linux peut être entravé. Cette option permet d’éviter les surcharges de l’usage d’une machine virtuelle. L'autre astuce importante à suivre pour que les utilisateurs améliorent leur expérience informatique est de faire une sauvegarde de leur ordinateur fréquemment sans échec afin d'éviter tout crash ou perte de données dû à WSL ou au double démarrage. Un outil appelé Mini Tool Partition wizard (outil de récupération de données pour restaurer des partitions) est également fortement recommandé pour gérer les partitions qui améliorent la vitesse de l'ordinateur si l'utilisateur choisit le double démarrage
